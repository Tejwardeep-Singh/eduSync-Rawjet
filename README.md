<h1 align="center">📚 EduSync – Smart School Management System</h1>

<p align="center">
EduSync is a full-stack school management system designed to simplify and automate administrative tasks such as student management, attendance tracking, marks management, and more. The platform provides role-based dashboards for Head, Teacher, and Student to ensure efficient and organized school operations.

## 🧠 Features

### 🔐 Role-Based Authentication

* Head, Teacher, and Student login systems
* Secure access with role-based authorization

### 👨‍🏫 Head Dashboard

* Manage teachers and students
* View overall school data
* Monitor performance and reports

### 👩‍🏫 Teacher Dashboard

* Manage student records
* Enter and update marks
* View class-wise data

### 🎓 Student Dashboard

* View marks and performance
* Access personal academic data

### 📊 Core Functionalities

* Student & Teacher Management (CRUD)
* Marks Management System
* Attendance & Datesheet Management
* Dynamic data updates without page reload

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/EduSync.git
```

2. Navigate to project folder

```bash
cd EduSync
```

3. Install dependencies

```bash
npm install
```

4. Create `.env` file and add:

```
MONGO_URI=your_mongodb_connection
CLOUDINARY_API_KEY=your_key
CLOUDINARY_SECRET=your_secret
JWT_SECRET=your_secret
```

5. Run the project

```bash
npm start
```
